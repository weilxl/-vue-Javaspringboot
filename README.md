# -vue-Javaspringboot
vue+Javaspringboot前后端分离项目
里面有网站端还有vue2写的小程序端
.
├── backend/            # 后端服务端代码 (Java/Spring Boot 或 PHP)
├── mobile-app/         # 移动端应用代码 (UniApp/小程序)
├── web-admin/          # 后台管理系统前端代码 (Vue.js/Element UI)
├── docs/               # 项目相关文档、设计图或 API 说明
├── .idea/              # IntelliJ IDEA 配置文件
├── .vscode/            # VS Code 配置文件
├── pom.xml             # Maven 项目配置文件
└── request.txt         # 接口请求记录或临时笔记文件
