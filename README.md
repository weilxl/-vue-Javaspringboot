# -vue-Javaspringboot
vue+Javaspringboot前后端分离项目
里面有网站端还有vue2写的小程序端
### 📂 项目目录结构

```text
├── backend/        # 后端服务端核心代码
├── mobile-app/     # 移动端应用 (UniApp/小程序)
├── web-admin/      # 后台管理系统前端 (Vue/Element UI)
├── docs/           # 项目开发文档与接口说明
├── .idea/          # IntelliJ IDEA 项目配置
├── .vscode/        # VS Code 编辑器配置
├── pom.xml         # Maven 依赖管理文件
└── request.txt     # 接口请求记录/测试笔记
