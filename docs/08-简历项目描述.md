# 个人效率管理系统 — 简历项目描述

---

**个人效率综合管理平台** — 项目负责人  
**2024.03 - 2025.06**  

**技术栈**：Vue2 | Element-UI | Vuex | Axios | ECharts | JWT | SpringBoot | MyBatis | MySQL | Git  

**项目介绍**：前后端分离的个人效率系统，实现待办、日程业务闭环，支持权限控制与任务数据可视化；PC+移动端双端入口，共用后端接口。  

**项目职责**：设计前端模块划分并封装可复用组件，使用 Vue2+ElementUI 开发待办与日程管理页，支持优先级、状态筛选及时间维度展示；统一封装 Axios 请求与 Token 鉴权、全局异常处理；Vuex 管理用户态，路由守卫控制登录与权限；ECharts 展示待办状态与优先级分布；参与后端待办、日程及统计接口开发（SpringBoot+MyBatis Plus+MySQL）。  

**项目成果**：形成任务与日程闭环，仪表盘支撑待办状态与优先级可视化；双端数据统一、多场景协同；权限清晰、结构规范，可维护与迭代能力良好。
